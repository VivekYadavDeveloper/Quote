## Run Locally

Clone the project

```bash
  git clone https://github.com/VivekYadavDeveloper/Quote.git
```

Go to the project directory

```bash
  cd quote_vault
```

Install dependencies

```bash
  flutter clean && pub get
```

Run the app

```bash
  flutter run
```
